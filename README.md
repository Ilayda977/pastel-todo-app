# 🌸 Pastel Todo App

A minimalist, responsive, and aesthetic Todo List application built to learn the fundamentals of React and TypeScript. 

## ✨ Features
- **Modern & Pastel UI:** Relaxing pastel tones with soft animations.
- **Type Safety:** Built completely using TypeScript interfaces.
- **Local Storage Integration:** Your tasks are saved automatically and never lost on page refresh.
- **Responsive Design:** Optimized for both mobile and desktop screens.

## 🛠️ Tech Stack
- **Framework:** React 18
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Custom CSS3 (Pastel Palette)
- **Deployment:** Vercel
